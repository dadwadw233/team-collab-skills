# Team Collab Protocol Pointer

Apply when the workspace has `obsidian-docs/`, team-collab references in `AGENTS.md`/`CLAUDE.md`, or the user asks about handoff, checkpoint, team docs workflow, TODO ownership, Feishu automation, project audit, or docs normalization.

Required behavior:

1. Read repo-root `AGENTS.md` first.
2. Read `obsidian-docs/CURRENT.md`, `RISKS.md`, `NEXT.md`, and `TODO.md` before substantive work.
3. Use `$handoff <topic>` for end-of-session handoff and `$checkpoint` for mid-session state snapshots.
4. Use the installed `team-collab-protocol` skill as the source of truth. Do not improvise partial handoff/checkpoint behavior from this pointer alone.
