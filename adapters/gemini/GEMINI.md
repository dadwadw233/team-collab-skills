# Team Collab Protocol Pointer

If this repository is a team project, follow repo-root `AGENTS.md` and the installed `team-collab-protocol` skill from `https://github.com/dadwadw233/team-collab-skills`.

Strong signals: `obsidian-docs/`, team-collab references in `AGENTS.md`/`CLAUDE.md`, or user requests about handoff, checkpoint, shared docs, TODO ownership, Feishu automation, project audit, or docs normalization.

Use `/handoff <topic>` or the prompt `$handoff <topic>` for end-of-session handoff. Use `/checkpoint` or `$checkpoint` for mid-session state snapshots.
