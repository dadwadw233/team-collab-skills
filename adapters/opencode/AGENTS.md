# Team Collab Protocol Pointer

This project uses the team-collab protocol when `obsidian-docs/` exists or the user asks about handoff, checkpoint, shared docs, TODO ownership, Feishu automation, project audit, or docs normalization.

Read the installed `team-collab-protocol` skill from `https://github.com/dadwadw233/team-collab-skills` and follow repo-root `AGENTS.md` as the source of truth.

Use `$handoff <topic>` for end-of-session handoff and `$checkpoint` for mid-session state snapshots.
